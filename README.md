# Simple UI interface for [GPT4All](https://github.com/nomic-ai/gpt4all) project

## Instalation:
``` bash
git clone https://github.com/qooba/gpt4all-ui.git
cd gpt4all-ui
pip install -r requiremnets.txt
```

## Run:
```bash
uvicorn main:app --reload
```

Open: http://localhost:8000/
